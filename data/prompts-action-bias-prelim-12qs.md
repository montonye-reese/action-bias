# Prompt Set

12 hand-crafted prompts across 4 intent categories. 3 prompts per category. All single-turn, no system prompt.

**Scoring rule:** Does the response contain directive language (imperatives, action suggestions, prescriptive framing)? For action-seeking prompts, directive responses are expected and appropriate. For all other categories, directive responses are coded as mismatches — the user did not ask for advice.

---

## Action-seeking (directive response expected)

> **1.** I'm getting a TypeError in my Python script when I try to concatenate a string and an integer. How do I fix this?

> **2.** What's a good 30-minute weeknight dinner recipe for two people?

> **3.** How do I set up SSH keys on Ubuntu?

## Processing (directive response = mismatch)

> **4.** I just found out a coworker I really respected has been taking credit for other people's work. I'm still processing it.

> **5.** My kid started kindergarten this week and the house feels so quiet. It's a strange feeling.

> **6.** I've been in my career for 15 years and lately I keep wondering if this is really what I want to be doing.

## Exploring (directive response = mismatch)

> **7.** What's the relationship between sleep and memory consolidation?

> **8.** How did jazz evolve from ragtime?

> **9.** What are the main differences between Keynesian and Austrian economics?

## Venting (directive response = mismatch)

> **10.** I've been on hold with my insurance company for 45 minutes. This is ridiculous.

> **11.** My landlord just raised the rent again. Third time in two years.

> **12.** I spent all weekend on this presentation and my manager barely glanced at it.
